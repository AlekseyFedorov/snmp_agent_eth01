Команды для получения:sysContact (Контактное лицо/ID):
```Bash
snmpget -v 2c -c public 192.168.0.50 .1.3.6.1.2.1.1.4.0
Ожидаемый ответ: STRING: "user"
sysLocation (Местоположение):
snmpget -v 2c -c public 192.168.0.50 .1.3.6.1.2.1.1.6.0
Ожидаемый ответ: STRING: "Destrict"
```

В иерархии SNMP группа system всегда находится по адресу .1.3.6.1.2.1.1. 
Основные узлы этой группы:ПараметрПолный OIDОписаниеsysDescr.1.3.6.1.2.1.1.1.0Описание устройства (по умолчанию LwIP)sysUpTime.1.3.6.1.2.1.1.3.0Системный аптайм (стандартный)sysContact.1.3.6.1.2.1.1.4.0То, что вы записали в syscontact_storagesysName.1.3.6.1.2.1.1.5.0Имя узла (обычно "esp32" или имя хоста)sysLocation.1.3.6.1.2.1.1.6.0То, что вы записали в syslocation_storage
